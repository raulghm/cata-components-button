# Cata components-button

`cata-components-button` components-button styles.

Read more about how to use [SUIT CSS](https://github.com/suitcss/suit/).

## Installation

## NPM 
```
npm install cata-components-button
```

## Yarn
```
yarn add cata-components-button
```
